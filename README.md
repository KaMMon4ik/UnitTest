<h1>UnitTest</h1>

<h3>Тестирование функций работы с массивами на языке программирвоания си</h3>

Файл test.c содержит 6 функций для тестирования:
- test_task_1 - тестирует функцию 1 задания из практической работы №16 <a href="https://sites.google.com/view/course-of-study1-c/практика/работа-16?authuser=0">(Ссылка)</a>
- test_insert_element - тестирует функцию вставки элемента в массив на определенный индекс
- test_delete_element - тестирует функцию удаления элемента под определенным индексом из массива
- test_sort_array - тестирует функцию сортировки массива от меньшего к большему
- test_sum_elements - тестирует функцию суммы элементов от первого до второго индекса
- test_find_element - тестирует функцию поиска первого элемента определенного значения

<h1>Тестирование функций</h1>
<h3>Для тестирования собственных функций с помощью данного кода необходимо скачать и перенести файлы в ваш проект. Файлы должны находиться в папке проекта (Там же, где находится файл, который необходимо протестировать)</h3>

<h3>Необходимо подключить заголовочный файл в тестируемый код:</h3>
<blockquote>#include "test.h"</blockquote>

<h3>В функции main вызывается необходимая функция для тестирования, в параметр функции передается название собственной функции:</h3>
<blockquote>test_task_1(task_1);<br>test_insert_element(insert_element);<br>test_delete_element(delete_element);<br>test_sort_array(sort_array);<br>test_sum_elements(sum_elements);<br>test_find_element(find_element);</blockquote>

<h3>В файле test.c можно изменять 10 строчку, отвечающую за количество тестов функции, и 11 строчку, отвечающую за максимальный размер массива при тестировании</h3>

<h1>Возможные ошибки при тестировании</h1>

Ошибки, которые могут появится:
- Функция тестирования может вызывать вашу функцию и передавать в нее пустой указатель на массив
- Функция тестирования может вызывать вашу функцию и передавать отрицательную длину массива
- Функция тестирования может вызывать вашу функцию и передавать неправильные индексы для вставки, удаления или диапазон индексов суммирования
- Функция сортировки должна возращать новый указатель на массив, не изменяя переданный указатель
- Функция поиска элемента должна возращать индекс первого найденного элемента

<h3>Ваши функции должны обрабатывать все случаи ошибок, в том числе если пользователь ввел в параметры функции неправильные данные, тогда необходимо возращать NULL либо 0. Если при тестировании программа прекращает работу, значит ваша функция не проверила корректность каких-либо параметров. Если функция работает неверно, то под номером теста появится описание ошибки</h3>
